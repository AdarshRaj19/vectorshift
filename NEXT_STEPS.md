# Immediate Next Steps (1-2 days implementation)

## Quick Wins
These are small but impactful features that can be added quickly:

### 1. Node Categories in Toolbar
- Group nodes by category (Input, Processing, Output, Logic)
- Collapsible sections for better organization
- Search/filter functionality in toolbar

### 2. Pipeline Statistics
- Real-time node/edge count in header
- Pipeline complexity score
- Execution time estimates

### 3. Enhanced Text Node
- Syntax highlighting for variables
- Auto-complete for variable names
- Rich text editing with formatting

### 4. Node Templates
- Save custom node configurations as templates
- Quick-apply templates to new nodes
- Share templates between pipelines

### 5. Mini-map Enhancements
- Zoom to fit functionality
- Node filtering in mini-map
- Customizable mini-map position
